Proyecto Customer Shopping Data - Análisis del Comportamiento del Cliente

Objetivo: Analizar el rendimiento comercial del dataset de compras, identificando patrones de consumo y categorías clave para la toma de decisiones estratégicas.

Herramientas utilizadas: SQL (Extracción y transformación de datos), Looker Studio (Procesamiento y creación de reportes visuales).

Proceso:
* Extracción: Creación de consultas SQL para obtener información relevante de ventas.
* Análisis: Segmentación de clientes, categorías y métodos de pago mediante consultas SQL.
* Visualización: Creación de dashboards y gráficos para representar los hallazgos principales.

Principales Hallazgos (Insights):
* Tendencia de Facturación: Se observa una evolución en los ingresos a lo largo del periodo analizado, lo que sugiere identificar los momentos de mayor actividad.
* Análisis por Categoría: Permite determinar qué categorías tienen mayor peso en la facturación total.
* Segmentación de Clientes: Identificación de patrones según el género y preferencias de pago.
