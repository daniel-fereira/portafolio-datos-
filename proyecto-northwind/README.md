# Proyecto Northwind - Análisis de Ventas

## Objetivo
Analizar las ventas de la base de datos Northwind mediante consultas SQL y un reporte visual.

## Herramientas utilizadas
- SQL
- Google Sheets

## Proceso
- Creación de consultas SQL para obtener información de ventas.
- Análisis de clientes, productos y categorías.
- Creación de gráficas para visualizar resultados.
