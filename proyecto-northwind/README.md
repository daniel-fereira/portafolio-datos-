Proyecto Northwind - Análisis de Ventas
Objetivo
Analizar el rendimiento comercial de la base de datos Northwind, identificando patrones de compra y categorías clave para la toma de decisiones estratégicas.

Herramientas utilizadas
SQL: Extracción y transformación de datos.

Google Sheets: Procesamiento y creación de reportes visuales.

Proceso
Extracción: Creación de consultas SQL para obtener información relevante de ventas.

Análisis: Segmentación de clientes, productos y categorías mediante consultas SQL.

Visualización: Creación de dashboards y gráficos para representar los hallazgos principales.

Principales Hallazgos (Insights)
Tendencia de Crecimiento: Se observa un crecimiento sólido en la facturación entre 1996 y 1997, seguido de un ajuste en 1998, lo que sugiere una oportunidad para investigar los factores que afectaron el rendimiento en ese último año.

Estacionalidad: El análisis mensual de 1997 revela una tendencia ascendente hacia finales de año, alcanzando picos en octubre y diciembre, lo cual es vital para la planificación de inventarios y logística.

Comportamiento por Categoría: Al comparar categorías como 'Bebidas' y 'Confites', logramos identificar qué productos tienen mayor rotación y cuáles presentan comportamientos más estables durante el año.
